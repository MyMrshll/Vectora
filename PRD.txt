# Product Requirements Document (PRD)

## RAG Management Web Application

---

# 1. Product Overview

## Product Name

**RAG Admin Platform**

## Product Type

Web-based CRUD application untuk mengelola sistem Retrieval-Augmented Generation (RAG).

## Purpose

Platform ini digunakan untuk:

* upload dan manajemen knowledge base,
* indexing dokumen ke vector database,
* monitoring embedding,
* query/testing RAG,
* dan administrasi model AI.

Target utama adalah mempermudah developer atau researcher dalam membangun dan mengelola pipeline RAG tanpa harus mengakses command line secara manual.

---

# 2. Background

Saat membangun sistem RAG, proses seperti:

* upload dokumen,
* chunking,
* embedding,
* indexing,
* query retrieval,
* dan monitoring vector store

biasanya dilakukan secara manual menggunakan script.

Hal ini menyebabkan:

* sulit debugging,
* sulit maintenance,
* tidak scalable,
* dan tidak user friendly.

Karena itu dibutuhkan dashboard web CRUD untuk mengelola seluruh lifecycle RAG.

---

# 3. Goals

## Primary Goals

* Mempermudah manajemen knowledge base RAG
* Menyediakan dashboard CRUD untuk dataset dan vector
* Memungkinkan testing retrieval secara realtime
* Mempercepat development AI assistant
* Mendukung local LLM maupun cloud LLM

---

# 4. Non-Goals

Versi MVP tidak mencakup:

* fine tuning model
* AI agent orchestration
* multi-region deployment
* distributed vector search
* enterprise billing
* marketplace/plugin system
